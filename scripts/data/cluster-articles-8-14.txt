Cursor AI কীভাবে ব্যবহার করবেন? বাংলা গাইড
কোডিং করতে গিয়ে কি মনে হয় যদি কেউ পাশে থেকে সাহায্য করত? Cursor AI ঠিক সেটাই করে। এটি একটি AI-ফার্স্ট কোড এডিটর যেখানে আপনি বাংলায় বা ইংরেজিতে বললেই কোড লিখে দেয়।
বাংলাদেশি ডেভেলপাররা Cursor ব্যবহার করে রিপোর্ট করছেন তাদের কোডিং স্পিড ২-৩ গুণ বেড়ে গেছে। নতুন ডেভেলপার থেকে শুরু করে সিনিয়র ইঞ্জিনিয়ার — সবার জন্যই এটি কাজের।
Cursor AI কী?
Cursor হলো VS Code-এর উপর ভিত্তি করে তৈরি একটি AI-পাওয়ার্ড কোড এডিটর। এটি দেখতে VS Code-এর মতোই কিন্তু ভেতরে Claude, GPT-4 এর মতো শক্তিশালী AI মডেল বিল্টইন।

Cursor এর সেরা ৫টি Feature
১. Tab Autocomplete
কোড লিখতে লিখতে Cursor পরের লাইন কী হবে সাজেস্ট করে। Tab চাপলেই সাজেশন accept হয়।
২. Ctrl+K — Inline Edit
কোনো কোড সিলেক্ট করুন, Ctrl+K চাপুন, বলুন কী করতে চান — Cursor সেই অনুযায়ী কোড পরিবর্তন করবে।
৩. Ctrl+L — Chat
Cursor-এর চ্যাট উইন্ডো খুলবে। আপনার পুরো প্রজেক্ট সম্পর্কে প্রশ্ন করুন, Cursor বুঝে উত্তর দেবে।
৪. Composer
একটি ফিচার লিখতে বলুন — Cursor একাধিক ফাইল একসাথে তৈরি বা এডিট করবে।
৫. Bug Fix
Error message কপি করুন, Cursor-এর চ্যাটে পেস্ট করুন — সমাধান পাবেন।
Cursor vs GitHub Copilot — কোনটা ভালো?

বাংলাদেশে AI Tools ব্যবহারের সম্পূর্ণ Beginner গাইড
AI tools এর কথা শুনেছেন, কিন্তু কোথা থেকে শুরু করবেন বুঝছেন না? এই গাইড আপনার জন্যই লেখা।
এখানে আমরা একেবারে শুরু থেকে ধাপে ধাপে বলব কীভাবে বাংলাদেশে AI tools ব্যবহার করবেন। কোনো পূর্ব অভিজ্ঞতা লাগবে না।
শুরু করার আগে যা জানতে হবে
AI tools কঠিন কিছু নয়। এগুলো ব্যবহার করা হোয়াটসঅ্যাপ বা ফেসবুক ব্যবহারের মতোই সহজ। শুধু একটু সময় দিয়ে পরিচিত হতে হবে।
লাগবে: স্মার্টফোন বা কম্পিউটার + ইন্টারনেট + ইমেইল একাউন্ট
লাগবে না: বিশেষ দক্ষতা, ক্রেডিট কার্ড (শুরুতে), VPN
ধাপ ১: সঠিক Tool বেছে নিন
আপনি কী করতে চান সেটা আগে ঠিক করুন। তারপর সেই কাজের জন্য সেরা tool বেছে নিন:
লেখালেখি করতে চান? → ChatGPT বা Claude
ছবি তৈরি করতে চান? → Ideogram বা Leonardo AI
কোড লিখতে চান? → Cursor বা Bolt.new
ভিডিও বানাতে চান? → Kling AI
ভয়েসওভার চান? → ElevenLabs
প্রেজেন্টেশন চান? → Gamma
পরামর্শ: একসাথে অনেক tool শিখতে যাবেন না। প্রথমে একটিতে দক্ষ হন, তারপর আরও শিখুন।
ধাপ ২: Gmail দিয়ে সাইন আপ করুন
বেশিরভাগ AI tool Google বা GitHub একাউন্ট দিয়ে সাইন আপ করা যায়। Gmail থাকলেই শুরু করা সম্ভব।

ধাপ ৩: প্রথম Prompt লিখুন
Prompt মানে হলো AI-কে যা করতে বলছেন তার নির্দেশনা। ভালো prompt মানে ভালো ফলাফল।
খারাপ Prompt
"কিছু লিখে দাও"
ভালো Prompt
"আমি একজন বাংলাদেশি ফ্রিল্যান্স ওয়েব ডেভেলপার। Upwork-এ একজন নতুন ক্লায়েন্টকে প্রথম ইমেইল লিখতে চাই। ক্লায়েন্ট একটি ই-কমার্স ওয়েবসাইট বানাতে চান। ইমেইলটি বাংলায় লিখো, পেশাদার টোনে।"
দেখুন দ্বিতীয় prompt-এ কে লিখছেন, কী লিখছেন, কার জন্য লিখছেন — সব বলা আছে।
ধাপ ৪: ফলাফল রিফাইন করুন
AI প্রথমবার পারফেক্ট ফলাফল না দিলে হতাশ হবেন না। বলুন কী পরিবর্তন করতে হবে:
"এটি আরও সংক্ষিপ্ত করো"
"বাংলায় অনুবাদ করো"
"আরও প্রফেশনাল করো"
"৩টি বিকল্প ভার্সন দাও"
